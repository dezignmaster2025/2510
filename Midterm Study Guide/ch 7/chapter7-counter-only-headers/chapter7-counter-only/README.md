This minimal project includes only **App** and **Counter** to match your assignment.

## Satisfies Requirements
1. `useState` inside `Counter` to store a number starting at `0`.
2. Current number displayed in a `<p>`.
3. Button increments the number by 1.
4. `title` prop passed from `App` to `Counter` and shown in an `<h2>`.
5. **Bonus:** `Reset` button sets number to `0`.

## Run
```bash
npm install
npm run dev
```
Open the printed URL (usually http://localhost:5173).
